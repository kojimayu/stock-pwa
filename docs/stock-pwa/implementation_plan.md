## 変更内容

### ① エアコン在庫テーブル（最も重要 — 画面上部に配置）

容量ごとの台数をテーブルで表示。エアコン管理テーブル（`AirconProduct`）から直接取得するため、材料マスタとの不整合は発生しない。

**表示イメージ:**

| 品番 | 容量 | 在庫 | 最低在庫 |
|-----|------|------|---------|
| RAS-AJ22 | 2.2kW | 4 | 2 |
| RAS-AJ25 | 2.5kW | 5 | 2 |
| RAS-AJ28 | 2.8kW | 13 | 2 |
| RAS-AJ36 | 3.6kW | 23 | 2 |

- 在庫 ≤ 最低在庫の行は黄色ハイライト
- 在庫 = 0 の行は赤ハイライト

### ② 発注状況セクション（材料・エアコン分離表示）

**材料発注**と**エアコン発注**を区別して表示。

```
【材料発注】
  発注#12  発注済  3品目   → 発注管理へ

【エアコン発注】
  AC-2026-001  発注済  RAS-AJ36 ×5   → エアコン発注管理へ
  AC-2026-002  一部入荷  RAS-AJ28 ×3   → エアコン発注管理へ
```

処理待ちがない場合は「処理待ちの発注はありません」と1行表示。

### ③ アラートのコンパクト化

テーブル形式を廃止 → **1〜2行のバナー形式**に変更。

```
🔴 在庫切れ 3件（ABC-001, DEF-002, GHI-003）→ 発注管理へ
🟡 在庫注意 5件（JKL-004, MNO-005 他3件）→ 商品管理へ
```

全て正常の場合:
```
✅ すべての材料在庫が十分です
```
→ 1行で済ませる。現在の大きな緑ボックス (アイコン + 2行テキスト + 大きなパディング) を廃止。

### ④ アラート種類（再検討結果）

| 種類 | 条件 | 表示色 | 表示形式 |
|------|------|--------|---------|
| 在庫切れ（材料） | 在庫=0、最低在庫設定あり、未発注 | 赤 | バナー＋品番リスト |
| 在庫注意（材料） | 0 < 在庫 < 最低在庫、未発注 | 黄 | バナー＋品番リスト |
| 在庫正常 | 上記に該当なし | 緑 | 1行テキストのみ |

> [!NOTE]
> 統計カード（画面上部4枚）は廃止します。
> エアコン在庫は容量別テーブル、発注は分離表示、アラートはバナー化するため、
> 統計カードに表示すべき数値がなくなります。

### ⑤ 最近のエアコン持出し — 縮小

- 5件 → 3件に削減
- コンパクトなリスト形式に変更
- 「すべて表示 →」リンクは維持
