# 管理者マニュアル（事務所用）

在庫管理システムの管理画面の使い方ガイドです。

---

## アクセス方法

ブラウザで以下にアクセス:
```
http://[サーバーIP]:3000/admin
```

---

## 1. 商品マスタ管理

**メニュー:** サイドバー → 商品マスタ

### 商品の追加
1. 「新規追加」ボタンをクリック
2. 必要情報を入力（商品名、カテゴリ、単価など）
3. 「保存」をクリック

### 商品の編集
1. 一覧から対象商品の「編集」ボタンをクリック
2. 情報を修正
3. 「保存」をクリック

### Excelインポート
1. 「インポート」ボタンをクリック
2. Excel/CSVファイルを選択
3. プレビューを確認して「実行」

---

## 2. 在庫管理

**メニュー:** サイドバー → 在庫管理

### 在庫確認
- 一覧で現在の在庫数を確認
- 最低在庫を下回ると赤色で警告表示

### 在庫調整
1. 対象商品の「調整」ボタンをクリック
2. 調整数と理由を入力
3. 「実行」をクリック

---

## 3. 取引履歴

**メニュー:** サイドバー → 取引履歴

### 履歴の確認
- 業者別、日付別でフィルタ可能
- 「戻す」ボタンで誤った取引を取消（在庫が復元される）

### CSVダウンロード
1. フィルタで対象期間を絞り込む
2. 「CSV」ボタンをクリック
3. **表示されているデータのみ**がダウンロードされる
   - **形式**: 1行1明細（全行に日時・業者名を記載）、商品コード付き

---

## 4. エアコン管理

### 在庫管理・棚卸
**メニュー:** サイドバー → エアコン在庫

- 機種ごとの倉庫在庫・業者持出数・総在庫を確認
- サフィックス（年度コード）を設定

#### 棚卸の実施
1. 「棚卸開始」ボタンをクリック（メモも入力可能）
   - 全機種のシステム在庫がスナップショットされます
2. 各機種の **実数** 欄に実際の在庫数を入力
   - 差異がある箇所は色付きで表示されます（緑: 過剰、赤: 不足）
3. 入力が完了したら「確定」ボタンをクリック
4. 確認ダイアログで差異サマリーを確認し、**確認者名** を入力
5. 「確定する」をクリック → **システム在庫が実数に一括更新**されます

> **中止:** 「中止」ボタンで棚卸を中止できます。在庫は変更されません。

> **履歴:** 「履歴」ボタンで過去の棚卸結果を確認できます。

### 持出しログ
**メニュー:** サイドバー → エアコンログ

- 誰がいつどの物件に持出したか確認
- **差戻し（返却）案件**: 赤色背景・取り消し線・「返却済」バッジで強調表示
- 「戻す」で在庫を復元
- CSVエクスポート可能

## 5. 業者管理

**メニュー:** サイドバー → 業者管理

### 業者の追加
1. 「新規追加」ボタンをクリック
2. 業者名とPINコード（初期値: 1234）を入力
3. 「保存」をクリック

### PINリセット
1. 対象業者の「編集」をクリック
2. PINを変更
3. 「保存」をクリック

---

## 6. 発注・入荷管理

**メニュー:** サイドバー → 発注管理

### 発注書の作成（自動）
1. 「発注候補を自動生成」をクリック
   - 在庫が「最低在庫数」を下回っている商品が自動的にリストアップされます。
2. 作成された「下書き (DRAFT)」の発注書をクリックして詳細を開く
3. 数量を確認・修正し、「発注を確定する」をクリック
   - ステータスが「発注済 (ORDERED)」に変わります。
   - **※ご注意**: この操作を行っても、**仕入先へ自動でメールやFAXは送信されません**。別途、電話やWeb等で実際の発注を行ってください。

### 発注書の作成（手動）
自動生成を使わず、手動で発注書を作成することもできます（例：電話発注分の登録など）。

1. 「手動作成」ボタンをクリック
2. 仕入先名を入力して「作成」をクリック
   - 空の発注書が作成されます。
3. 詳細画面で、「商品を追加」欄から商品名や品番で検索
4. 商品を選択するとリストに追加されます
5. 数量を入力して「発注を確定する」をクリック

### 入荷処理（検品）
発注した商品が届いたら、以下の手順で在庫に反映させます。

1. 「発注済」または「一部入荷」の発注書を開く
2. 商品ごとの「入荷操作」欄を確認
3. 実際に届いた数量を入力し、「確定」をクリック
   - 入荷した分だけ**在庫数が自動的に加算**されます。
4. すべての商品が入荷されると、ステータスが「入荷完了 (RECEIVED)」になります。

## 6.5 エアコン発注（メール送信）

**メニュー:** サイドバー → エアコン管理 → 発注管理

エアコンの発注書（PDF）を自動生成し、メールで送信する機能です。

### 発注の作成
1. 「新規発注」ボタンをクリック
2. **納品先拠点** を選択（プラスカンパニー本社、ZION、草津中庄 等）
3. 商品ごとに **発注数量** を入力
4. 必要に応じて **備考** を入力
5. 「下書き保存」をクリック
   - 発注番号が自動採番されます（例: AC-2026-001）

### PDFの確認
1. 下書き状態の発注で「PDF確認」ボタンをクリック
2. 注文書PDFがダウンロードされます
3. 内容を確認してください

### 発注メールの送信
1. 下書き状態の発注で「発注メール送信」ボタンをクリック
2. 確認ダイアログで内容を確認
3. 「送信する」をクリック
   - 注文書PDFが添付されたメールが日立担当者に送信されます
   - CCに設定された関係者にも同報されます
   - ステータスが自動的に「発注済 (ORDERED)」に変わります

### 入荷チェック
1. 「発注済」または「一部入荷」の発注で「入荷チェック」ボタンをクリック
2. モーダルに各商品の発注数・入荷済み数・残数・**現在在庫**が表示されます
3. **全数入荷**: 「全数(X)」ボタンで残数をセット → 「入荷確定」をクリック
4. **一部入荷**: 数量欄に実際の入荷数を入力 → 「入荷確定」をクリック
   - 入荷した分だけ**在庫数が自動的に加算**されます
   - 残りがある場合はモーダルが更新されます
5. すべて入荷すると「入荷完了 (RECEIVED)」になり、モーダルが自動で閉じます

### 発注のキャンセル
- **下書き（DRAFT）**: 「🗑️ 削除」ボタンで永久削除（確認ダイアログあり）
- **発注済（ORDERED）**: 「⊘ キャンセル」ボタンでキャンセル（確認ダイアログあり、履歴は保持）

### メール設定の変更
1. 発注管理画面右上の「メール設定」ボタンをクリック
2. **送信先（To）**: メールの宛先（日立担当者）
3. **CC（同報先）**: 関係者のメールアドレスを追加・削除
4. **差出元会社名**: メール件名に表示される自社名
5. **納品先拠点**: 拠点の追加・無効化・削除
6. 「メール設定を保存」をクリック

## 7. よくある操作

| やりたいこと | 操作 |
|-------------|------|
| 今日の取引を確認 | 取引履歴 → 日付フィルタで今日を選択 |
| 在庫切れ商品を確認 | 在庫管理 → 「在庫なし」でフィルタ |
| 月次レポート作成 | 取引履歴 → 月でフィルタ → CSV |
| 誤入力を修正 | 取引履歴 → 「戻す」で取消 |
| エアコンを発注する | エアコン発注管理 → 新規発注 → メール送信 |
| 入荷を記録する | エアコン発注管理 → 入荷チェック |
| メール送信先を変更 | エアコン発注管理 → メール設定 |
