# 業者マニュアル（現場タブレット用）

現場で材料・エアコンを持出す際の操作ガイドです。

---

## ログイン

1. QRコードをカメラにかざす、または業者名を選択
2. PINコード（4桁）を入力
3. 「ログイン」をタップ

> **初回ログイン時**: PIN変更を求められたら、新しいPINを設定してください

---

## 材料を持出す

### 1. モード選択
ログイン後、「**材料出庫**」をタップ

### 2. 商品を選ぶ
1. 左側のカテゴリから選択
2. 商品一覧から必要な商品をタップ
3. 数量を入力して「追加」

#### 🔍 検索
- 検索欄にキーワードを入力して絞り込めます
- 検索欄横の **🎤マイクボタン** をタップすると音声で入力できます

> **⚠️ 音声入力の制限:** 音声入力はブラウザのセキュリティ制約により、**`https://` または `localhost`** でのみ動作します。  
> 現在のシステムは `http://192.168.x.x` 形式のLAN接続のため、**音声入力は使用できません**。

### 3. 手入力（一覧にない商品）
1. 画面左下の「**＋ 手入力商品を追加**」をタップ
2. 商品名と数量を入力
3. 「追加」をタップ

### 4. 確定
1. 右上のカートで内容を確認
2. 「**確認へ進む**」をタップ
3. 最終確認して「**持出し確定**」

---

## エアコンを持出す

### 1. モード選択
ログイン後、「**エアコン出庫**」をタップ

### 2. 物件を検索
1. 左側の「管理No検索」に管理番号（6桁）を入力
2. 🔍ボタンをタップ
3. 物件情報が表示される

### 3. エアコンを追加
1. 右側の「機種選択」から該当するkWをタップ（例: 2.2kw）
2. 持出しリストに追加される
3. 複数台の場合は繰り返す

### 4. 確定
1. 「**持出し確定**」（緑のボタン）をタップ
2. 完了メッセージが表示される

---

## 履歴を確認する

### 材料履歴
1. 材料出庫画面 → 右上の「**履歴**」をタップ
2. 自分の持出し履歴が表示される

### エアコン履歴
1. エアコン出庫画面 → 右上の「**履歴**」をタップ
2. 入力ミスがあれば「**修正**」で訂正可能

---

## 画面の更新

データが反映されない場合:
1. **3本指でタップ**してメニューを開く
2. 「**Go to Start URL**」をタップ
3. 画面が再読み込みされる

---

## 困ったとき

| 症状 | 対処法 |
|------|--------|
| ログインできない | PINが正しいか確認。分からなければ事務所に連絡 |
| 商品が見つからない | 「手入力」で商品名を直接入力 |
| 間違えて持出した | 履歴画面で事務所に連絡（取消は事務所で対応） |
| 画面が固まった | 3本指タップ → Go to Start URL |
| 音声入力ができない | LAN接続（http://IPアドレス）では使用不可。キーワード入力で代用 |
