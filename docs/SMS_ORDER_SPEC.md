# SMS発注仕様書

承認日: 2026-02-18

## フォーマット

```
#[発注番号]
[品番] [商品名] [数量][単位]
[品番] [商品名] [数量][単位]
...
```

### 例

```
#1234
VVF203 VVF2mm×3芯 1巻
KB360 化粧ブロック 2箱
CT14 CT-14 5個
```

## ルール

- 1行目: `#` + 発注番号のみ
- 2行目〜: 品番 + 半角スペース + 商品名 + 半角スペース + 数量 + 単位
- 空行なし、挨拶なし、宛名なし、敬称なし
- 70文字以内 = 3.3円（3〜4品が目安）

## 料金目安（ドコモ国内SMS）

| 商品数 | 文字数目安 | 料金 |
| :--- | :--- | :--- |
| 1〜4品 | 〜70文字 | 3.3円 |
| 5〜8品 | 〜134文字 | 6.6円 |
| 9品超 | 135文字〜 | 9.9円〜 |
