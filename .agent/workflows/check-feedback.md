# 会話開始時のフィードバック確認

新しい会話を開始したら、最初に以下を行う：

1. `docs/FEEDBACK.md` を確認する
2. 「未対応」セクションに項目があれば、ユーザーに作業予定として提案する
3. 対応完了後は該当項目を「対応済み」セクションに移動する

## 提案フォーマット例

```
docs/FEEDBACK.md に以下の未対応項目があります：

1. [!] 商品一覧のモバイル表示が崩れる
2. 代理入力で日付選択後にリセットされる

どれから対応しますか？
```
